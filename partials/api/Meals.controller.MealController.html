MealController
controller in module Meals

Description

Controller component "mealPage". Allows the user to add meal to cart, delete meal, edit meal and etc.

Dependencies

  • MealsFactory
  • CartFactory

Methods

  • addMeal()

    Call method addMealToCartList owned CartFactory.

  • removeMeal()

    Call method removeMeal owned CartFactory and method deleteModifiers owned CartFactory.

  • saveMeal()

    Call method removeMeal owned CartFactory and method addMealToCartList owned CartFactory.

Properties

  • currentMeal

    Current meal.

  • currency

    Currency.

  • selectAmount

    Current meal select amount.

  • mealStatus

    Current meal status.

  • cartCount

    Count meal in cart.
